这是一个用于制作 AppStore 预览视频的桌面小工具。

只需要选择一张图片，设置视频时长和目标文件大小，工具就会自动生成一个全程展示该图片的 MP4 视频。视频会保持原图尺寸，并采用 H.264 编码和静音 AAC 音频轨，适合用于 App Store 预览素材制作。

默认参数为 25 秒、450MB，可根据需要自行调整。工具支持Windows、 Intel Mac 和 Apple Silicon Mac。

使用方法：

Windows：
  
  解压Windows版本后运行StillVideoMaker.pyw，首次制作会自动下载ffmpeg.exe，请耐心等待完成即可；

Mac：
  
  解压Mac版本直接点击运行app即可。
